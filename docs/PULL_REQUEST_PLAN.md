# Pull Request Plan

These local branches are ready to push and open as real GitHub pull requests.

| Branch | Pull Request Title | Purpose |
| --- | --- | --- |
| `feature/better-search-ranking` | Improve task search ranking | Adds reusable ranking helpers for task search results. |
| `feature/project-dashboard-report` | Add project dashboard summaries | Adds per-project dashboard data and markdown output. |
| `feature/task-import-export` | Expand import and export utilities | Adds JSON-friendly task movement helpers. |
| `feature/workflow-automation-rules` | Add workflow automation rules | Adds helpers for blocked, active, and completed workflow movement. |
| `feature/calendar-planning-view` | Add calendar planning view | Adds week planning helpers for upcoming tasks. |

## How To Open Them

1. Push `main`.
2. Push one feature branch.
3. Open a pull request from that branch into `main`.
4. Review and merge it on GitHub.
5. Repeat with the next branch.
